## Introduction
This messaging app allows you to send and receive messages in real-time.

## Installation
To install the app, use the following commands:

```bash
git clone https://github.com/davidba1234/messaging-app.git
cd messaging-app
npm install
```

## Usage
To start the app, run:

```bash
npm start
```
